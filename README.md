# Content Management App by Redux/JavaScript

## Description

This is a simple app to manage content. It is built with React, Redux, JavaScript, and Material-UI. It is a work in progress. for ACC Assignment

## Using Technologies

- React
- Redux
- TypeScript
- Material-UI
- React Router
- React Router Redux
- Redux Thunk

## Features

- Add content
- Edit content
- Delete content
- View content
- Search content
